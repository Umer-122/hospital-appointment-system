About This Project

The Hospital Appointment Management System is a full-stack web application built as a Final Semester Project. It digitizes the process of booking and managing hospital appointments, reducing manual work and improving efficiency.


Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: FastAPI (Python)
  • Database: MongoDB Atlas
  • Deployment: Render + Vercel
